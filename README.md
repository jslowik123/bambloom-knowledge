# Bambloom Knowledge Base

This repository is the **single source of truth** for autonomous AI agents creating UGC and marketing videos for **Bambloom: 75 Soft Challenge**.

## What is Bambloom?

Bambloom is a wellness companion app for the **75 Soft Challenge** — a 4-habit, 75-day wellness journey designed as a compassionate alternative to "75 Hard". The app emphasizes **progress over perfection**, featuring an adorable panda mascot, a flower garden reward system, daily affirmations, and reflection journaling.

## How to Use This Repo

1. **Clone this repo** to get all brand knowledge
2. **Clone the main Bambloom project** to access binary assets (images, animations)
3. **Read `brand/identity.md` first** to understand the brand
4. **Read `marketing/ugc-guidelines.md`** for video creation rules
5. **Reference `content/`** for copy and messaging to use in videos
6. **Check `video-templates/`** for proven scripts to adapt

## Key Brand Rules

- **Tone:** Encouraging, warm, non-judgmental — like a supportive friend
- **Never:** Harsh, demanding, perfectionist, or guilt-inducing
- **Always:** Celebrate progress, embrace rest days, normalize imperfection
- **Aesthetic:** Soft, feminine, warm pastels — "nice girlie look"
- **Mascot:** Adorable panda — always kind, never pushy
- **Motto:** "Progress over perfection"
