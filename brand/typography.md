# Typography

## Primary Fonts

### Quicksand (Headers & Display)
- **Source:** Google Fonts
- **Style:** Rounded, friendly, warm
- **Weights Used:** 400 (Regular), 600 (SemiBold), 700 (Bold)
- **Usage:** All headings, display text, prominent UI labels, branding text
- **Letter Spacing:** 0.1-0.5px for visual warmth
- **Why:** Rounded letterforms feel approachable and feminine â€” matches the "girlie" aesthetic

### Inter (Body Text)
- **Source:** Google Fonts
- **Style:** Clean, highly readable
- **Weights Used:** 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)
- **Usage:** Body text, descriptions, form fields, smaller UI elements
- **Why:** Excellent readability at small sizes, professional but friendly

## Video Typography Guidelines

- **Headlines in videos:** Quicksand Bold, large size (40-60px equivalent)
- **Body text in videos:** Inter or Quicksand Regular
- **Emphasize key words** with bold weight or color change (not ALL CAPS unless for hook)
- **Hook text** can use larger, bolder treatment â€” Quicksand Bold with text shadow
- **CTA buttons:** Quicksand SemiBold, rounded pill shape
- All text should have sufficient contrast against backgrounds
- Use text shadows on busy backgrounds for readability
