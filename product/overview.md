## Key Features

### 1. Beautiful Journey Path Visualization
- Track all 75 days on a visual path
- See completed, incomplete, and current days
- Milestones marked at key points (Day 7, 14, 21, 30, etc.)

### 2. Adorable Panda Mascot
- Personal cheerleader that adapts to your progress
- 9+ mood states (happy, celebrating, encouraging, gentle, sleepy, proud, etc.)
- Context-aware messages (time of day, progress, milestones, comebacks)
- Dynamic dialogue system with 50+ unique messages

### 3. Flower Garden Reward System
- Every completed day plants a flower
- 12 regular flower varieties (random daily)
- 3 special Golden Bloom varieties (for perfect days)
- Golden spiral layout for natural aesthetic
- Beautiful visual progress over 75 days

### 4. Daily Reflection Journal
- Mood tracking with star ratings
- 6 guided reflection prompts per day
- Private journaling space
- Encrypted local storage

### 5. Daily Affirmations
- 78 affirmations across 8 categories
- Personalized to challenge day
- Categories: Growth, Self-Love, Strength, Confidence, Mindfulness, Resilience, Transformation, Gratitude

### 6. Progress Analytics
- Streaks and perfect day counter
- Completion rates per habit
- Weekly progress tracking
- Most consistent habit identification

### 7. Additional Features
- Home screen widget for quick access
- Daily notification reminders
- 3 color themes (Warm Earth, Serene Garden, Blush Dream)
- Light and dark mode
- Works fully offline
- Optional cloud sync via Supabase
- Cross-platform (iOS, Android, Web, macOS, Windows)

## Tech Stack (for reference)

- **Frontend:** Flutter (Dart)
- **State Management:** Riverpod
- **Local Storage:** Hive (encrypted)
- **Backend:** Supabase (auth + optional sync)
- **Auth:** Google Sign-In, Sign in with Apple
- **Animations:** Lottie, flutter_animate, Confetti
- **Video/Screenshots:** Remotion (React/TypeScript)
- **Platforms:** iOS 14+, Android 6+, Web, macOS, Windows
